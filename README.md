# Back In Past 🕰️

[![Deployed on Vercel](https://img.shields.io/badge/Deployed_on-Vercel-black?logo=vercel)](https://back-in-past.vercel.app/)
[![GitHub top language](https://img.shields.io/github/languages/top/AyushCOMET/Back-In-Past)](https://github.com/AyushCOMET/Back-In-Past)

**[Back In Past](https://back-in-past.vercel.app/)** is an interactive computer project dealing with the history of British India. It explores the early days of foreign trade, the arrival of the English East India Company, and the major wars fought by the invading British forces against native Indian powers.

## 🌍 Live Demo

Check out the live website here: **[back-in-past.vercel.app](https://back-in-past.vercel.app/)**

## 📚 Historical Events Covered

The project provides detailed accounts and navigation for the following historical milestones:
* **The Battle of Plassey**
* **The Battle of Buxar**
* **The Anglo-Mysore Wars** (First, Second, Third, and Fourth)
* **Science Day / C.V. Raman**

## 🛠️ Tech Stack

This project is built using core frontend web technologies:
* **HTML** (76.8%) - For semantic structure and content delivery.
* **CSS** (20.5%) - For styling, layouts, and responsive design.
* **JavaScript** (2.7%) - For interactive elements and routing logic.

## 📂 Repository Structure

* `index.html` — The main landing page featuring the origins of foreign trade and the British arrival.
* `battle1.html` & `battle2.html` — Dedicated pages for the Battle of Plassey and Battle of Buxar.
* `anglo1.html` to `anglo4.html` — Detailed breakdowns of the four Anglo-Mysore Wars.
* `cvraman.html` — A special page dedicated to National Science Day.
* `assets/` — Contains images, styles, and other static resources.
